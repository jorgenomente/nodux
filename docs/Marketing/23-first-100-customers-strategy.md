# NODUX — First 100 Customers Strategy

Versión: v1.0
Estado: Fundacional
Propietario: NODUX
Última actualización: 2026

---

# Propósito de Este Documento

Este documento describe la estrategia para conseguir los primeros 100 clientes de NODUX.

Los primeros clientes son críticos porque permiten:

- validar el producto
- perfeccionar la narrativa de ventas
- construir casos de éxito
- generar reputación en el sector

El objetivo no es crecer rápido.

El objetivo es construir **una base sólida de clientes satisfechos**.

---

# Principio Estratégico

Los primeros 100 clientes no llegan mediante marketing masivo.

Llegan mediante:

conversaciones directas  
implementaciones exitosas  
referencias entre dueños de tiendas.

---

# Segmento Inicial

El nicho ideal para los primeros clientes es:

Dietéticas y tiendas de productos naturales.

Características del nicho:

- inventario complejo
- productos con vencimiento
- rotación constante
- dependencia de procesos manuales
- dueños muy involucrados en la operación

Este nicho tiene exactamente los problemas que NODUX resuelve.

---

# Estrategia Geográfica

Concentrar los primeros clientes en una zona específica.

Ejemplo:

Buenos Aires  
↓  
Palermo / Belgrano  
↓  
tiendas del nicho.

Esto facilita:

visitas presenciales  
instalación rápida  
soporte cercano  
referencias entre tiendas.

---

# Estrategia de Conversaciones

El objetivo inicial es simple:

hablar con tiendas todos los días.

Ejemplo de ritmo semanal:

30 conversaciones con tiendas  
10 demos  
3 nuevos clientes

Este ritmo permite construir una base inicial rápidamente.

---

# Identificación de Tiendas Ideales

Señales de que una tienda es buen cliente para NODUX:

• múltiples sucursales  
• inventario visible en estanterías  
• gran variedad de productos  
• pedidos manuales a proveedores  
• presencia online desconectada  

Estas tiendas suelen tener dolor operativo claro.

---

# Conversación Inicial

La conversación no debe empezar vendiendo software.

Debe empezar con el problema.

Ejemplo:

“Estoy hablando con tiendas que tienen problemas para controlar inventario, vencimientos y pedidos a proveedores cuando tienen varias sucursales.

¿Cómo manejan eso ustedes hoy?”

Esto abre una conversación natural.

---

# Demo de Conversión

La demo debe durar:

10–15 minutos.

Mostrar principalmente:

POS  
inventario por sucursal  
productos por vencer  
sugerencias de compra  
control de caja  
pedidos online

El objetivo es mostrar **flujo operativo conectado**.

---

# Estrategia de Piloto

Para reducir fricción se recomienda comenzar con:

1 sucursal piloto.

Duración:

30 días.

Objetivo:

demostrar impacto real en la operación.

Métricas típicas:

- menos productos vencidos
- inventario más confiable
- cierres de caja más claros
- pedidos a proveedores más organizados

---

# Conversión a Cliente Completo

Si el piloto funciona:

expansión a otras sucursales.

Esto permite crecer dentro de cada cliente.

Ejemplo:

cliente con 3 sucursales  
↓  
empieza con 1  
↓  
expande a 3.

---

# Estrategia de Referencias

Las referencias entre dueños de tiendas son extremadamente poderosas.

Después de implementar NODUX:

pedir recomendaciones.

Ejemplo de pregunta:

“¿Conoces otras tiendas que tengan estos mismos problemas?”

Una referencia suele tener mayor probabilidad de conversión.

---

# Construcción de Casos de Éxito

Cada cliente satisfecho se convierte en:

testimonio  
caso de estudio  
contenido educativo.

Ejemplo de caso:

“Cómo una cadena de dietéticas redujo pérdidas por vencimientos.”

Estos casos ayudan a cerrar nuevos clientes.

---

# Objetivo de los Primeros 100 Clientes

Cuando NODUX alcanza 100 clientes, el negocio cambia.

Se obtiene:

reputación sectorial  
flujo constante de referencias  
casos de éxito documentados  
mensaje de ventas validado.

A partir de ese punto el crecimiento puede acelerarse.

---

# Métrica Clave

La métrica más importante no es:

número de clientes.

Es:

**número de sucursales operando con NODUX.**

Esto refleja impacto operativo real.

---

# Declaración Final

Los primeros 100 clientes de NODUX no se consiguen con publicidad.

Se consiguen con algo más poderoso:

conversaciones reales con dueños de tiendas.

Cada conversación es una oportunidad para demostrar que el retail necesita algo más que software.

Necesita **infraestructura operativa**.

Y esa infraestructura es NODUX.
